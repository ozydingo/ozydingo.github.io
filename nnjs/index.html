nnjs

Welcome to the nnjs ("neural nets in javascript", pronounced "ninjas", because duh)!

This lightweight demo of neural nets in action is meant to foster some intuitive understandings of neural net training. It is not fast, performant code!! But I hope you'll find the home-grown js code approachable, readable, and instructive. The intent is to educate and illustrate how simple, at their core, these techniques can be!

This is (and will always be) a work in progress. Currently, the network is training on fixed data representing a NAND function

Hit the "play" () button to get started!

Click or right-click on the output graph (bottom-right) to add custom training data points

Coming soon(ish):

  • Visualization of validation data
  • User-inputted watch points (table of values will display)
  • Activation function selection and visualization
  • Visualization of learning slowdown, RELU death, etc.
