Welcome to the nnjs (pronounced "ninjas", because duh) splash page!

This is a lightweight demo of neural nets in action! This demo is meant to foster some intuitive understandings of neural net training on a simple, user-inputtable (TODO) data set. Hit the "play" () button to get started!

This is (and will always be) a work in progress. Currently, the network is training on fixed data representing a NAND function

Coming soon(ish):

  • User-inputted data points (binary data only)
  • User-inputted watch points (table of values will display)
  • Customizeable network layers (num layers, num neurons)
